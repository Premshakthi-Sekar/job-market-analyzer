# Job Market Analyzer

This project analyzes job descriptions and predicts the likely job role while extracting key skills from the text.

## Goals
- Extract important skills from job descriptions
- Classify job roles such as Data Analyst, Data Scientist, and ML Engineer
- Build a simple app to demonstrate the model

## Project Structure
- `data/raw/` - raw job description dataset
- `notebooks/` - notebooks for exploration and experiments
- `src/` - source code
- `app/` - application code
- `tests/` - test files
